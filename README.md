# Dormigram

A dormitory central system for requests and more (coming soon...).

## Run Locally

Clone the project

```bash
  git clone https://github.com/mestrey/dormigram.git
```

Go to the project directory

```bash
  cd dormigram
```

Install dependencies

```bash
  composer install
```

Start the server on public directory
